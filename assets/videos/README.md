# assets/videos/

Place large media files (videos, audio) in this folder. Suggestions:

- Use kebab-case for filenames: `module-intro.mp4`, `lesson-1.mp4`.
- Prefer MP4 (H.264) for widest compatibility.
- Keep file sizes reasonable for offline use; consider creating low/medium/high quality variants.
- Reference videos from pages like: `<video src="/assets/videos/lesson-1.mp4" controls></video>`.

Tip: If you add many videos, create subfolders (e.g. `assets/videos/module-1/`).
